# Learning Journey

This repo is my personal learning journey as a self-taught software engineer. It's a living record of growth — capturing experiments, exercises, projects, and concepts I'm working through as I continue to level up.

There's no finish line here. The goal is consistent progress: writing better code, understanding fundamentals more deeply, and building things that push my current limits.

## What You'll Find

- Code experiments and practice problems
- Mini-projects built while learning new technologies
- Notes and takeaways from things I'm studying
- Iterations and improvements over time

## Philosophy

I believe the best engineers are the ones who never stop learning. This repo reflects that — it won't always be perfect, and that's the point. Every commit is a step forward.
